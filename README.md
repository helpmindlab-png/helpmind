# Mind Spark (데모 소스)

이 ZIP은 지금까지 대화에서 확정한 방향(감성 톤, 3개 테스트, 연쇄 추천 카드, 3개 완료 후 프로파일, 이미지 저장, **개인 데이터 서버 저장 없음**)을 기준으로 만든 **정적(Static) 프론트 데모**입니다.

## 포함 기능
- 테스트 3개 (각 8문항, 1~5 선택)
  - 사회적 에너지
  - 감정 회복 속도
  - 인간관계 피로도
- 선택 즉시 다음 문항으로 자동 이동(완료율 최적화)
- 결과 화면: **짧은 설명 + “자세히 보기”**
- 다음 테스트 **카드 제안(사용자 클릭 유도, 자동 시작 없음)**
- 3개 완료 후 **에너지 프로파일(Chart.js 레이더 차트)**
- 결과/프로파일 **이미지(PNG) 저장**
- “끝내기(흔적 지우기)” 버튼(메모리 상태 초기화)

## 프라이버시/보안(중요)
- 이 데모는 **로컬스토리지/세션스토리지/IndexedDB에 응답/점수/결과를 저장하지 않습니다.**
- 점수/응답은 **메모리(변수)에서만** 유지됩니다.
- 새로고침/탭 종료 시 데이터는 사라집니다.
- 공용 PC에서는 시크릿 모드를 권장합니다.

### 운영 배포 시 권장 헤더
테스트/결과/프로파일 경로에 아래 헤더를 적용하면 브라우저 캐시/뒤로가기 재노출 위험을 크게 줄일 수 있습니다.

- `Cache-Control: no-store, no-cache, must-revalidate, max-age=0`
- `Pragma: no-cache`
- `Expires: 0`

## 실행 방법
- `index.html`을 더블클릭해서 브라우저로 열면 됩니다.
- 또는 간단한 로컬 서버로 실행하면 더 안정적입니다.
  - 예: `python -m http.server 8000` 후 `http://localhost:8000`
